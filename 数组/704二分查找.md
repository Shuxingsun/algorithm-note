# 二分查找

## 题目

> 给定一个 n 个元素有序的（升序）整型数组 `nums` 和一个目标值 target  ，写一个函数搜索 `nums` 中的 target，如果目标值存在返回下标，否则返回 -1。

**实例1**

```
输入: nums = [-1,0,3,5,9,12], target = 9
输出: 4
解释: 9 出现在 nums 中并且下标为 4
```



## 解答

- 左闭右开
- 左开右闭

```cpp
int search(vector<int>& nums, int target) {
        // 二分法查找,左闭右开区间
        int left = 0;
        int right = nums.size();
        // 因为是左闭右开，所以需要 <
        while(left < right){
            // 防止溢出
            int middle = left+(right-left)/2;
            if (nums[middle] > target){
                //主要改动的地方，因为是右开，不可取
                right = middle;
            }else if(nums[middle] < target){
                left = middle + 1;
            }else{
                return middle;
            }
            
        }
        return -1;
    }
```

```cpp
int search(vector<int>& nums, int target) {
        // 二分法查找,左闭右闭区间
        int left = 0;
        int right = nums.size()-1;
        // 因为是左闭右闭，所以可以 <=
        while(left <= right){
            // 防止溢出
            int middle = left+(right-left)/2;
            if (nums[middle] > target){
                //主要改动的地方
                right = middle - 1;
            }else if(nums[middle] < target){
                left = middle + 1;
            }else{
                return middle;
            }
            
        }
        return -1;
    }
```
